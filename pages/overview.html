什么是 Claude Code

Claude Code 是 Anthropic 官方出品的 CLI 工具，让开发者可以在终端中与 Claude 协作完成软件工程任务——编辑文件、执行命令、搜索代码库、协调复杂工作流。

它不是一个简单的聊天包装器，而是一个完整的 Agentic 系统：Claude 可以主动调用工具、递归执行多步任务、管理子 Agent、并在上下文耗尽时自动压缩对话。

核心能力

  • 文件操作：读取、写入、编辑（精确字符串替换）、搜索文件内容
  • Shell 执行：在用户 shell 环境中运行任意命令，支持后台执行
  • Web 访问：抓取网页内容、执行搜索
  • Agent 编排：生成子 Agent 处理独立子任务，支持并行执行
  • MCP 集成：连接外部 MCP 服务器，扩展工具能力
  • 上下文管理：自动压缩超长对话，保持会话连续性
  • Skill 系统：用户可定义可复用的工作流，通过 /skill-name 调用
  • IDE 集成：通过 Bridge 系统与 VS Code、JetBrains 双向通信
